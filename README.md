# 📌 Projeto Pizzas JS

> Aplicação web para simulação de pedidos de pizzas, desenvolvida com JavaScript puro, focada em manipulação de DOM e lógica de interface.

---

## 🧠 Contexto

Este projeto foi desenvolvido no início dos meus estudos em JavaScript, com base em um curso prático, onde implementei e adaptei funcionalidades para reforçar o aprendizado.

O objetivo foi entender, na prática, conceitos fundamentais como:
- manipulação de DOM
- eventos
- controle de estado no frontend

> ⚠️ Projeto de estudo com base em tutorial, contendo adaptações e melhorias próprias.

---

## 🚀 Tecnologias utilizadas

- JavaScript (Vanilla JS)
- HTML
- CSS
